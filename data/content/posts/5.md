- 按住 "Windows+R" 键, 输入 regedit" 然后 "Ctrl+Shift+Enter" 键以管理员模式运行。

- 根据以下路径进入注册表:

```reg
HKEY_CURRENT_USER -> Software -> Microsoft -> Terminal Server Client -> Default
```

- 找到你需要删除的记录删除即可。
